      真的很抱歉让你生气，但是我希望我们还是朋友。那天晚上我准备了很多话想跟你说，但是事实上我表述的很糟糕，情绪思绪也很混乱，事实证明我的情绪化真的很严重。从9月开始，其实我就察觉到感觉有点不对劲了，说话越来越少，节假日也没经常一起出去，准确来说，如果我再醒目一点点，可能从你说你很忙很忙的时候我就应该想到关系不会维持很久。现在这份工作对你来说确实身心疲惫，每天上下班烟尘滚滚，课程还排得满满。可能这个时候你身边更加需要的是一个的是能轻易见面，能跟你分担你的不开心的人，可是这些我当时都没有做到，甚至周末回去见面也没有做到，我当时确实忽略了，没能做到我能做到的。我不知道你当时下班哭着回家的时候有没有这样问过自己：同样是找对象，为什么要找个异地的，想见又不能随时见，什么事都要自己面对。但是我觉得你有这样问过，这个或许真的异地困难的缩影。我不知道你有没有奇怪我为什么察觉到而不去挽留，其实我是有努力过的，我努力的方向就是进大厂，然后尽快回广州，我不确定这个时间有多长，所以我一直不敢承诺你什么时候可以回广州，我也知道这样一直拖着其实挺不负责任的，但是我没有办法，我只能尽快的尝试进大厂，但是很遗憾失败。可能你觉得我的想法和做法很奇怪，这也证明我真的是一个自卑感很强的人（很讽刺喔，之前一直跟你说积极乐观，但是自己却这么自卑），成年人之间谈恋爱不单单是感情这么简单，还要有物质和未来（你可以说我很现实，但是事实就是这样的），没有足够的条件之前我不敢给你任何承诺。但是我又很喜欢你，不想放弃。这里面的矛盾真的令我很烦恼。
      我也有问过我自己认识你究竟是对是错，我也没有答案，对的一方面从6月到8月底，我真的很开心，我目前不确定你有没有喜欢过我，但是我一定有喜欢过你，而且确实尝到了恋爱的滋味，每天睡觉前说晚安，起床第一件事就是打开手机看聊天记录。那种聊天的感觉就是你不会担心对方觉不觉得你烦，反正聊什么大家都能聊下去。中间有段时间我虽然很忙，但是我重来不觉得维持这段感情很累，甚至有种感觉就是工作虽累，但是生活不累，这或许就是我跟你不一样的点，工作虽累，但是每天能跟喜欢的人在一起，哪怕只是简简单单的聊天，都是自己支撑下去的动力。错的一面是这段感情分开的时候确实很痛，虽然做好了心里准备，但是仍然很痛，我到现在仍然会失眠，早上很早醒，中午还是睡不着，我真的不知道还会持续多久，庆幸的时候，现在晚上能很快入睡（生物钟不被打破的话）。
      我之前对你的好都是出自真心的，也没有要求说我对你好，就一定要你对我好。我可能说错了很多气话，也做了些意气用事的事，包括那天说做朋友是为了买保险，也包括我真的有想过考虑你表嫂介绍给我的那个美女，我真的很幼稚。但是我冷静下来之后，我想清楚了，没放下之前我不应该开展一段新感情的，不能毁了自己，也不能害了别人。同时我也想清楚自己现在首要做的事情是什么了。最后我真的希望我们目前还是朋友，希望以后大家见面了你很能把我当成朋友，而不是形同陌人。

# 抱歉 # 幼稚 # 矛盾
