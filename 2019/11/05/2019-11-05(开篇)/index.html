2019-11-05(开篇)

发表于 2019-11-05 | 分类于 日记 |

      没想过自己会开一个博客专门写日记，我承认我真的很想挽留，我每天都有很多话想跟你说，但是我知道我现在找你聊天就是死缠难打，这反而让你更累，现在的我不知道做什么才是对的，我只能将一切交给时间了，每天记录点点滴滴，或许能打动你，或许会慢慢忘记你。写日志是我想到的对你影响最小的方式了，也是我最后的办法了。我不确定什么时候会发个博客地址给你，可能是立刻发给你，可能在考试的日期（虽然没有考试了，但是我还是想尽力去答题），可能永远不会。但是即使我发了，你也不会看，就算你看，也不一定会回心转意，甚至你可能会觉得我很烦，但是我尽力就问心无悔了。也或许我把这个博客当成自己忘记你，放下你的一个过程，或许在我不更新博客的时候，我才能说得上正式放下了。。。。。。
