发表于 2019-11-05 | 分类于 日记 | | 阅读次数:

      没想过自己会开一个博客专门写日记，我承认我真的很想挽留，我每天都有很多话想跟你说，但是我知道我现在找你聊天就是死缠难打，这反而让你更累，现在的我不知道做什么才是对的，我只能将一切交给时间了，将每天想到的话记录下来。或许能打动你，或许会慢慢忘记你。写日志是我想到的对你影响最小的方式了，也是我最后的办法了。我不确定什么时候会发个博客地址给你，可能是立刻发给你，可能在考试的日期（虽然没有考试了，但是我还是想尽力去答题），可能永远不会。但是即使我发了，你可能也不会看，就算你看，也不一定会回心转意，甚至你可能会觉得我很烦，但是我尽力就问心无悔了。也或许我把这个博客当成自己忘记你、放下你的一个过程，或许在我不更新博客的时候，我才能说得上正式放下了。。。。。。但是我真的很希望你能看到
      你说你很累，我能理解，所以我不想加重你的负担，所以看不看的选择权还是在你手里。你说我们不合适，但是你对我的了解又有多少呢？我对你的了解可能也不多，所以我不知道我们是不是真的不合适，也不知道这个合适到底有多重要。现在你已经没有时间和心情去想很多事情了，我也错过了很多机会。但是我知道我还喜欢你，我真的不想轻易放弃。我最近在想，我是不是用错方法，其实去大厂是不是这么重要？经济条件是不是真的这么重要？还是说珍惜眼前人才是最重要的！如果我当时多点陪你，我们线上聊天少了，我可以每周回去找你，多点时间陪你，甚至可以提前回广州。。。。。。或许结局就不会这样了没，我记得你说过你看重的是每一天，是每一天，可是已经没有如果了，每次想到这些我都非常后悔。或许这个就是成长或者改变，我真的希望在我忘记你之前，你会觉得我们也可以很合适。。。。。。
