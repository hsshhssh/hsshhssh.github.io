       今天特别棒，睡眠质量提升了，虽然5点钟醒了一次，但是断断续续还是睡到8点了。可是今天中午还是睡不着，不知道什么时候才能睡回午觉，希望这个到来的时间不会很久吧。今天我翻看聊天记录，谈心的那一段，那次应该是我们第一次谈心。你说我这么久了都没有问你想要什么，然后后面谈到，你是极度缺乏安全感的人，只有实实在在的人和事才能给你带来安心的感觉，如果继续异地，以后开心的不能即使分享，难过不能在身边安慰，闹点小事在手机里表达不清楚变成大事…….可能要折磨死你了。我听到的时候其实也是有点伤感的，我也和你一样，但是我知道这个事实目前没法改变，起码今年没法改变。但是我最错的是没把这件事情时刻记在心里，也许我时时刻刻提醒自己，可能不会让你提醒和暗示一次次失望。我现在真的很想再挽留你一次，明天周五，不知道会不会忍不住回广州再“逼”你出来聊天，即使挽留不住，起码堂堂正正，大大方方的say goodbye，做回朋友。我今天不断的安慰自己，我感觉的得到，那段谈心的聊天，我们当时聊的时候，你肯定是希望我们后面能继续下去的，不然你也不会跟我聊这些。我就在想，会不会是现在情况变，你工作上确实很烦，每天都很累，我工作也很烦，很累，或许5个月的约定对现在的你来有点措手不及，一乱就想歪，然后给出各种不合适的理由。不如你告诉我我是不是自己欺骗自己吧。其实是不是5个月不重要，5个不要不够就一年，一年不够就两年，关键是要你想清楚，到底是真的不合适，还是时间不够了。

# 好转 # 谈心
